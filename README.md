# 📝 Real-time Character Counter

A simple and responsive web application that allows users to type a message and view the character count in real time. It displays the number of characters typed, the remaining characters, and prevents users from exceeding the maximum limit.

## 🚀 Live Demo

 https://ishu122004.github.io/Real-time-Character-counter/

## 📂 GitHub Repository

 https://github.com/ishu122004/Real-time-Character-counter

## ✨ Features

- Real-time character counting
- Displays typed characters
- Displays remaining characters
- Maximum character limit of 200
- Prevents typing beyond the limit
- Error message when the limit is reached
- Clean and responsive user interface
- Built using Tailwind CSS

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (DOM Manipulation)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ishu122004/Real-time-Character-counter.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Preview

Type inside the text area to see:
- Characters typed
- Characters remaining
- Warning when the character limit is reached
